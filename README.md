# scm-ui-vue

[vue3](https://v3.cn.vuejs.org/) + [antd-design-vue](https://www.antdv.com/docs/vue/introduce-cn) + [vxe-table](https://vxetable.cn/v4/#/table/start/install)

## install and use

- yarn

```yarn
yarn add scm-ui-vue
```

- use

```yarn
import ScmUI form 'scm-ui-vue'

app.use(ScmUI)
```
